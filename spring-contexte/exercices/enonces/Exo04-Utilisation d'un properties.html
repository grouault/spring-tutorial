1 Utilisation d'un fichier properties

Création du fichier properties

En continuant à partir de l'exercice 3.

Créez un fichier properties à la racine de votre répertoire spring, donnez-lui le nom de adresses.properties.

Définissez des adresses dans votre fichier properties:

# Format : clef=valeur
# Information pour la premiere adresse
addr1.codePostal=75000
addr1.ville=Paris
addr1.pays=France
addr1.rue=14 rue de la gloire
# Information pour la seconde adresse
...

Modification du fichier XML Spring

Faites-en sorte de charger le fichier de properties via la classe Spring PropertyPlaceholderConfigurer.

Déclarez simplement le bean dans votre fichier Spring. 

<bean class="org.springframework.beans.factory.config.PropertyPlaceholderConfigurer">
  <property name="location" value="adresses.properties"/>
</bean>

Modifiez la déclaration de vos beans adresses afin qu'elle face usage du contenu du fichier de properties. Faites usage de ${xxxx} où xxxx est le nom de la clef.

Respectez bien le nom de vos clefs. 

...
<bean id="ad2" class="com.exo.Adresse">
  <property name="codePostal" value="${addr2.codepostal}"/>
  <property name="rue" value="${addr2.rue}"/>
  <property name="ville" value="${addr2.ville}"/>
  <property name="pays" value="${addr2.pays}"/>
</bean>
...

Test

Lancez votre classe Main et regardez ce qui s'affiche.

En cas d'erreur lisez attentivement les messages Spring, n'hésitez pas à scroller. Avez-vous bien respecté vos noms de clefs ?.
