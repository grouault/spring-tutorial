1 Beans

Création de la classe Adresse

En continuant à partir de l'exercice 1.

Dans le répertoire src, créez une classe Java Adresse.

Faites un clic droit sur src, puis nouveau/classe

  • package : com.exo
  • nom : Adresse
  • superclass : java.lang.Object

Ajoutez un constructeur sans paramètre dans votre classe.

Ajoutez des attributs privés :

  • String codePostal
  • String rue
  • String ville
  • String pays

Surchargez la méthode toString afin d'avoir une représentation claire de vos instances.

Codez/générez les get/set pour chaque propriété (codePostal, rue, ville, pays).

Ajoutez un constructeur avec paramètre (codePostal, rue, ville, pays) dans votre classe.

Modification de la classe Client

Reprenez votre classe Client, ajoutez lui un attribut de type com.exo.Adresse.

Codez/générez le get/set pour cette nouvelle propriété.

Ajustez votre/vos constructeurs afin d'y inclure l'adresse.

Ajustez votre méthode toString afin d'y inclure l'adresse.

Modification du fichier XML Spring

Déclarez un bean Adresse ayant comme propriétés votre adresse

Liez-le à votre bean client. Faites usage de <property> dans votre <bean>


Créez un second bean Client.

Créez un second bean Adresse en faisant usage du constructeur (<constructor-arg>).

Créez un troisième bean Client et essayez de le relier à une adresse dans une même déclaration.

Test

Récupérez vos beans clients et affichez-les via le log.

Lancez votre classe Main et regardez ce qui s'affiche.

En cas d'erreur lisez attentivement les messages Spring, n'hésitez pas à scroller.
